Cloud-hosted and highly scalable

Cloud scalability in cloud computing refers to the ability to increase or decrease IT resources as needed to meet changing demand

Mostly Cross-platform

Web applications are typically described as cross-platform because, ideally, they are accessible from any web browser due to the factor that browser is the ultimate platform for any website. Web applications generally employ a client–server model, but vary widely in complexity and functionality.

Modular and loosely coupled

A loosely coupled system is one in which components are weakly associated (have breakable relationships) with each other, and thus changes in any one component least affect the existence or performance of another component.

Tested with automated tests

Automated testing is a process that validates if software is functioning appropriately and meets all its requirements before it is released into production. This software testing method uses scripted sequences that are being executed by testing tools.

User-friendly navigation

User Friendly navigation implies ease of access and hassle-free functioning when the user is visiting the website or web page. A web navigation only gets successful if it is easy to handle, navigate and access
